# Calculadora
Calculadora Feita com DOM , JavasCript e comentada !!
Calculadora com DOM e JavaScript
Este é um projeto de calculadora simples feita com DOM e JavaScript.

Funcionalidades
A calculadora é capaz de realizar as quatro operações básicas de matemática (adição, subtração, multiplicação e divisão) e também é capaz de calcular o resultado de expressões com múltiplas operações e parênteses.

Como funciona
A calculadora usa o DOM (Document Object Model) para manipular elementos HTML e JavaScript para realizar os cálculos. Quando um botão é clicado, um evento é acionado e a função correspondente é chamada para processar a entrada do usuário.

A entrada do usuário é armazenada em uma string que é analisada e avaliada pela função eval() para gerar o resultado.

Como usar
Clone este repositório para o seu computador.
Abra o arquivo index.html em seu navegador.
Use os botões da calculadora para inserir os números e operações desejados.
Pressione o botão = para obter o resultado.
Comentários
O código está devidamente comentado para explicar cada trecho de código e sua funcionalidade. As variáveis e funções têm nomes significativos para facilitar a compreensão do código.

Algumas precauções foram tomadas para garantir a segurança do usuário, como evitar a avaliação de expressões maliciosas usando a função eval() e limitar o número máximo de dígitos na tela da calculadora para evitar sobrecarga de memória.
